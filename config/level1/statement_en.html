Welcome to the School of Magic

  The Goal

Disarm Maester Wizard before he kills you. His magic is really powerful and the only way to defeat him is to use his own spell against him.

  Rules

At the start of the game, you are attacked by the Maester Wizard spell. You have to read this spell from standard input and use the same spell against him. That means, you have to write the read spell on the standard output.
 
Victory Conditions
Use the same spell as the Maester Wizard
 
Lose Conditions
Provide incorrect spell or no spell at all

  Game Input

In each turn, your program must read single text from standard input and print exactly the same text to standard output. Don't forget that your output has to end with newline
Input

Single line: spell, the Maester wizard is casting against you

Output
A single line that contains only spell. It have to be same spell Maester casts.
